## 🧠 About

I’m **Ali Firas (thesmartshadow)** — a **Security Researcher** focused on **source-code auditing**, **vulnerability research**, and **responsible disclosure**.

I ship real fixes: **clear reports**, **reproducible PoCs**, and **PRs with regression tests**.

**Research focus**
- Open-source security & supply-chain defense  
- Input validation, injection, and logic flaws  
- Patch engineering + regression tests  
- Security advisories & coordinated disclosure
